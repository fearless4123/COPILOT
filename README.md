## Proje Hakkında
Bu proje, GitHub Copilot'ta keşfedilen RCE (Uzaktan Kod Çalıştırma) zafiyetini (CVE-2025-53773) incelemektedir. Proje, Vanilla JavaScript kullanılarak statik bir şekilde geliştirilmiş olup, zafiyetin adım adım nasıl sömürüldüğünü gösteren interaktif bir simülasyon sunmaktadır.
